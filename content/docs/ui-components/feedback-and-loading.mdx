## Skeleton

Placeholder block with pulse animation:

```tsx
import { Skeleton } from "@/shared/components/ui/skeleton";

<div className="space-y-2">
  <Skeleton className="h-4 w-[200px]" />
  <Skeleton className="h-4 w-[160px]" />
</div>
```

## Toasts (Sonner)

The app uses [`sonner`](https://sonner.emilkowal.ski/) for notifications. `shared/components/providers/providers.tsx` mounts `<Toaster />` from `sonner` at the root so `toast()` works anywhere under `Providers`.

```tsx
import { toast } from "sonner";

toast.success("Saved");
toast.error("Something went wrong");
toast("Loading…", { duration: 2000 });
```

> **Note:** The repo also includes `shared/components/ui/sonner.tsx`, a themed `Toaster` wrapper (uses `next-themes` and custom icons). The default root layout uses the plain `Toaster` from `sonner`; switch to the wrapper if you want those styles globally.

## Spinner

Accessible loading indicator using `Loader2Icon`:

```tsx
import { Spinner } from "@/shared/components/ui/spinner";

<Spinner className="size-6" />
```

The component sets `role="status"` and `aria-label="Loading"`.

## Progress

Determinate progress bar (`value` 0–100):

```tsx
"use client";

import { Progress } from "@/shared/components/ui/progress";

<Progress value={33} />
```

## Alert

Inline status message (not a toast):

```tsx
import { AlertCircleIcon } from "lucide-react";
import { Alert, AlertDescription, AlertTitle } from "@/shared/components/ui/alert";

<Alert>
  <AlertCircleIcon />
  <AlertTitle>Heads up</AlertTitle>
  <AlertDescription>
    Your session will expire soon.
  </AlertDescription>
</Alert>
```

Use `variant="destructive"` for error styling. Place an icon as the first child for the grid layout to align title and description.
