---
title: Clone the repository
description: Get a local copy of the SaaS starter with Git.
---

## Clone with HTTPS

```bash
git clone https://github.com/YOUR_ORG/saas-starter.git
cd saas-starter
```

Replace the URL with the real remote for your fork or upstream repository.

## Clone with SSH

If you use SSH keys with your Git host:

```bash
git clone git@github.com:YOUR_ORG/saas-starter.git
cd saas-starter
```

## Shallow clone (optional)

To save bandwidth and disk when you only need the latest commit:

```bash
git clone --depth 1 https://github.com/YOUR_ORG/saas-starter.git
cd saas-starter
```

## Verify the checkout

```bash
git status
```

You should see a clean working tree on your default branch (for example `master` or `main`), depending on how the remote is configured.

> **Note:** If you plan to contribute upstream and also maintain your own fork, configure a second remote (see [Updating the codebase](./updating-the-codebase)) after cloning.
