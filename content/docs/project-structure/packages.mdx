Versions below are taken from **`package.json`** at the time of writing; run `pnpm list <pkg>` or open `package.json` for the exact semver ranges in your checkout.

## Framework and runtime

| Package   | Role |
| --------- | ---- |
| `next`    | **16.2.x** — App Router, dev server (`pnpm dev` uses `--turbopack`). |
| `react` / `react-dom` | **19.2.x** — UI runtime. |
| `typescript` | Type-checking (devDependency). |

```json
"next": "16.2.1",
"react": "19.2.4",
"react-dom": "19.2.4"
```

## Database

| Package           | Role |
| ----------------- | ---- |
| `@prisma/client`  | Generated client for PostgreSQL. |
| `prisma`          | CLI (migrations, generate, studio) — devDependency. |
| `@prisma/adapter-pg` | Prisma driver adapter for `pg`. |
| `pg`              | Node PostgreSQL driver. |

Schema is split under `prisma/models/*.prisma` with a root `prisma/schema.prisma` datasource using `provider = "postgresql"`.
