# SaaS Starter Documentation

Welcome to the documentation for the SaaS Starter.

This starter is built to help you launch quickly with a solid production-ready base.

## What is included

- Authentication
- Billing
- Dashboard
- Settings
- Documentation-ready structure
- Scalable feature-based organization

## Quick start

Follow these steps to get started:

1. Clone the repository
2. Install dependencies
3. Configure your environment variables
4. Run the development server

```bash
pnpm install
pnpm dev
```
