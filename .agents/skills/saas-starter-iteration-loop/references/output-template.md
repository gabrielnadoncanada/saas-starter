# Output Template

Use this structure when converting an audit into execution guidance.

## Audit Verdict
One short paragraph on whether the audit is accurate and well-prioritized.

## Reprioritized Fixes
For each fix include:
- title
- severity
- buyer impact
- smallest correction

## Agent Prompt
Write a direct implementation prompt that:
- names the outcome
- lists the exact changes to make
- lists what must remain untouched
- forbids unnecessary abstractions
- asks for a concise patch summary
