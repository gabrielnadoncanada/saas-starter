# How to Edit Public Content

## Purpose

Change public-facing text, links, and calls to action.

## Files to Edit

- `app/(marketing)/page.tsx`
- `app/(marketing)/pricing/page.tsx`
- `app/(auth)/sign-in/page.tsx`
- `app/(auth)/sign-up/page.tsx`
- `app/not-found.tsx`

## Steps

### Step 1 - Update the home page

Edit `app/(marketing)/page.tsx`.

### Step 2 - Update pricing copy

Edit the pricing page route and `PricingSection`.

### Step 3 - Update auth page copy

Edit the sign-in and sign-up route files.

### Step 4 - Update fallback pages

Edit `app/not-found.tsx` so the product tone stays consistent.

## Common Mistakes

- Forgetting the auth pages when rebranding the public surface
- Leaving starter copy in the 404 page
