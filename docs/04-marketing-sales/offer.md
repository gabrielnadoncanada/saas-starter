# Offer

## Purpose
Define the sellable offer, packaging, pricing direction, and commercial boundaries.

## Working offer concept
A premium one-time purchase AI-native SaaS starter that gives solo founders and small technical teams a monetization-ready base they can understand and customize quickly.

## Core promise
Stop rebuilding the same SaaS foundation. Start from a monetization-ready base you can actually understand, customize, and ship.

## Product framing
This is a premium starter built on a modern stack baseline. It should feel current and credible, not like a discounted boilerplate or a legacy code pack.

## Proposed packages
### Starter — 149 USD
- core codebase
- setup docs
- customization docs
- auth + billing + dashboard foundation
- workspace foundation
- landing page and pricing page

### Pro — 249 to 299 USD
- everything in Starter
- additional modules or premium blocks
- extended docs
- more advanced examples or patterns
- selected AI-native modules if they clearly increase perceived value
- stronger template coverage for common B2B SaaS scenarios

### Agency / Extended — 499 to 699 USD
- broader usage rights
- agency or client project usage terms
- priority support or onboarding options
- commercial rights aligned with client delivery use cases

## Pricing rules
- Never go below 149 USD.
- 149 USD is the entry price, not the ceiling.
- The pricing ladder should naturally direct serious buyers toward Pro.
- Higher tiers must justify price through saved time, broader rights, and clearer commercial payoff.

## Revenue model options
### Primary recommendation
One-time purchase with optional paid upgrades.

### Optional expansion
- paid support add-on,
- setup consultation,
- premium module packs,
- yearly updates plan.

## Support policy hypothesis
- documentation-first,
- paid premium support for higher tiers,
- avoid unlimited support promises in low tiers.

## Licensing questions
- single product usage or multiple products?
- solo vs team vs agency rights?
- update duration included?

## Open questions
- what is the cleanest Starter vs Pro split without weakening Starter at 149 USD?
- should AI modules live only in Pro or partly in Starter?
- should updates be lifetime or time-boxed?
