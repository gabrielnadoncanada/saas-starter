# Launch Overview

## Purpose
Turn the current repo into a commercially clear product that can be sold at a minimum of 149 USD without overpromising.

## Launch stance
Sell the product the repo already proves:
- modern Next.js 16 starter
- authentication foundation
- protected dashboard foundation
- Prisma/Postgres baseline
- Stripe-ready billing foundation
- pragmatic code organization that is easier to customize than overbuilt starters

Do not lead with unproven claims such as:
- complete AI-native platform
- full enterprise multi-tenant system
- exhaustive admin suite
- everything-you-need boilerplate rhetoric

## Commercial objective
Launch with a clear Starter plan at 149 USD and push serious buyers toward Pro.

## Core principle
The first launch wins by clarity, speed, and credibility.
It does not win by promising the broadest feature list.

## Files in this section
- `headline-options.md` — headline and subheadline options
- `pricing.md` — pricing ladder and feature split
- `features.md` — saleable feature inventory and proof framing
- `faq.md` — objection handling
- `launch-checklist.md` — execution checklist before launch
- `landing-page-copy.md` — conversion-oriented landing page copy draft
- `README-sales-outline.md` — public README structure for repo/product page

## Non-negotiables
- never price below 149 USD
- do not claim features not clearly present or intentionally shipping in the launch version
- keep copy confident but anti-hype
- optimize for trust and fast buyer comprehension
