# Persona

- Visiteur / prospect: voit le marketing, pricing et entre dans le funnel.
- Utilisateur final standard: utilise le produit du client dans son workspace.
- Owner du workspace: gère membres, invitations, billing, upgrades et organisation.
- Membre invité: rejoint un workspace via invitation.
- Admin plateforme: gère tous les users et toutes les organisations via l'admin interne.
- Acheteur du starter: fondateur ou petite équipe technique qui achète le starter pour lancer son SaaS.
