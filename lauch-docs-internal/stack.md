# Stack

Authentification: Better Auth
Multi-tenancy: Better Auth (Organization Plugin)
Billing: Better Auth (Stripe Plugin) + in-house one-time top-up billing
Front-end: Next.js 16
UI runtime: React 19
Language: TypeScript
Styling: Tailwind CSS v4
Component library: shadcn/ui
Data table: TanStack Table
Form schema validation: Zod
Database: PostgreSQL
ORM: Prisma
URL params synchronization: Nuqs
AI: Vercel AI SDK
AI component library: AI Elements
Storage: Unstorage
Queries management: TanStack Query
Email: Resend + React Email
Docs: Fumadocs
Testing: Vitest + Playwright
