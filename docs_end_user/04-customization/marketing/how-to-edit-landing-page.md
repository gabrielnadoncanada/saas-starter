# How to Edit Landing Page

## Purpose

Customize the public home page copy and section content.

## Files to Edit

- `app/(marketing)/page.tsx`
- `app/terminal.tsx`

## Steps

### Step 1 - Change the headline and CTA

Edit `app/(marketing)/page.tsx`.

### Step 2 - Change marketing sections

That same page currently contains the feature blocks and final CTA section.

### Step 3 - Change the terminal hero if needed

Edit `app/terminal.tsx` if you want to replace the terminal demo visual.

## Common Mistakes

- Looking for a CMS-like content layer that does not exist
- Changing only text and forgetting outbound CTA links
